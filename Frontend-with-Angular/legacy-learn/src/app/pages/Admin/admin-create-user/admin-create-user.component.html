Welcome, admins! This page empowers you to invite new users to join the Legacy Learn platform. By granting access to qualified individuals, you can expand the reach of Legacy Learn and empower more students on their academic journeys.

To grant a user access to Legacy Learn, simply enter their email address in the field below. Once their email is added to the approved list, they will receive an invitation to sign up for an account.
